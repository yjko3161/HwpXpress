# HwpXpress

윈도우에서 ".hwp" 파일을 ".hwpx"로 일괄 변환하는 프로그램입니다. 한글(Hancom Office) 프로그램을 자동으로 제어하여 변환을 수행합니다.

## 📋 사전 요구사항 (Prerequisites)

이 프로그램은 **한글(Hancom Office)** 프로그램이 설치된 윈도우 환경에서만 작동합니다.
- **Python 3.10 이상**: [Python 다운로드](https://www.python.org/downloads/)
- **한글 2010 이상** (권장: 한글 2018 이상)

## 🚀 설치 및 환경 설정 (Installation)

가상 환경을 자동으로 생성하고 필요한 라이브러리를 설치하기 위해 아래 단계를 따르세요.

1. **프로젝트 폴더로 이동**
   CMD 또는 PowerShell을 열고 프로젝트 폴더로 이동합니다.

2. **환경 설정 스크립트 실행**
   `setup_env.bat` 파일을 더블 클릭하거나 터미널에서 실행합니다.
   ```cmd
   setup_env.bat
   ```
   이 스크립트는 다음 작업을 수행합니다:
   - `.venv` 라는 가상 환경 생성
   - `pip` 업그레이드
   - `requirements.txt`에 명시된 필수 라이브러리(`pywin32` 등) 설치

## 💻 실행 방법 (Usage)

환경 설정이 완료되면 프로그램을 실행할 수 있습니다.

1. **가상 환경 활성화 및 실행**
   CMD 창에서 다음 명령어를 입력합니다. (아직 활성화되지 않은 경우)
   ```cmd
   .venv\Scripts\activate
   python main.py
   ```

2. **GUI 사용**
   - **Target Path**: 변환할 파일 하나 또는 폴더를 선택합니다.
     - `Browse File`: HWP 파일 하나 선택
     - `Browse Folder`: HWP 파일이 들어있는 폴더 선택 (하위 폴더 포함)
   - **Start Conversion**: 변환 시작 버튼을 누릅니다.
   - **Logs**: 하단 로그 창에서 진행 상황과 성공/실패 여부를 확인할 수 있습니다.
   - 변환된 `.hwpx` 파일은 원본 파일과 같은 위치에 저장됩니다.

## 🛠️ 주요 기능

- **개별/일괄 변환**: 파일 하나만 변환하거나 폴더 내의 모든 HWP 파일을 일괄 변환할 수 있습니다.
- **로그 기록**: 변환 성공, 실패, 건너뜀 등의 이력이 GUI와 `app.log` 파일에 남습니다.
- **예외 처리**: 암호 걸린 파일이나 손상된 파일 등으로 인한 오류 발생 시 로그를 남기고 다음 파일로 진행합니다.

## ⚠️ 주의사항

- 변환 중에는 **한글 프로그램 창이 열릴 수 있습니다.** 작업 중에는 마우스나 키보드 조작을 최소화하는 것이 좋습니다.
- 한글 보안 모듈 승인 창이 뜰 경우 '접근 허용'을 선택해야 자동화가 가능합니다. (보안 승인 모듈 등록이 필요할 수 있음)
